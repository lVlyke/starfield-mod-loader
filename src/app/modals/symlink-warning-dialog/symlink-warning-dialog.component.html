Symlinks are not enabled Starfield Mod Loader does not currently have permission to create symlinks. Some functionality will be unavailble.

To enable symlinks, follow the guide below:
Enable symlinks (Windows) To enable symlinks in Windows, you can either A) run Starfield Mod Loader as administrator (not recommended) or B) enable Windows Developer Mode. To enable Developer Mode, open the Windows "Settings" app, go to "For developers", and then enable "Developer Mode". Once enabled, Starfield Mod Loader should now be able to use symlinks.
